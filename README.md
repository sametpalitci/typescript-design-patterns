### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/sametpalitci/typescript-design-patterns
   ```
2. Go to the pattern folder
   ```sh
   cd singleton/ # e.g.
   ```
3. Compile file with typescript:
   ```sh
   ts-node index.ts
   ```
4. Happy Coding
